# Oaiso
===

![Sample](docs/img/oaiso_sample.png)

Slackで寿司の支払いを記録・参照できます。
Record the payment of SUSHI by Slash commands of Slack.

# Functions
- 記録（Record）
- 参照（Reference）
- クリア（Clear）
